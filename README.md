## 🚀 How to Clone and Run

**1. Clone the repository**
```bash
git clone https://github.com/Akshat090803/gushwork-assignment.git
```

**2. Navigate into the project folder**
```bash
cd gushwork-assignment
```

**3. Run locally**

No build step or dependencies needed. Open `index.html` directly in your browser, or use a local server:

```bash
npx serve .
```

Then open `http://localhost:3000` in your browser.
